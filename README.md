# Senzory a komunikační periferie na mobilních platformách

## Abstrakt
<p align="justify">Bakalářská práce se zaměřuje na&nbsp;rešerši existujících nejrozšířenějších senzorů a&nbsp;komunikačních periferií mobilních zařízení. Cílem je zmapovat a&nbsp;popsat jejich hardwarové a&nbsp;softwarové možnosti, včetně případné emulace, používaných komunikačních protokolů, potřebných oprávnění a&nbsp;limitů jednotlivých platforem. Na&nbsp;teoretickou část, která je zaměřena na&nbsp;platformy Android a&nbsp;iOS, navazuje návrh a&nbsp;implementace ukázkové mobilní aplikace pro&nbsp;operační systém Android, která demonstruje praktické využití vybraných senzorů a&nbsp;periferií. Aplikace je realizovaná s&nbsp;důrazem na&nbsp;srozumitelnost, přehlednost, možnou budoucí rozšiřitelnost, a&nbsp;má potenciál sloužit pro&nbsp;potřeby výuky v&nbsp;oblasti vývoje aplikací pro&nbsp;mobilní zařízení.</p>
